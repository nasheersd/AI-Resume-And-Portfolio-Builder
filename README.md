# AI-Resume-And-Portfolio-Builder Description :-
AI Resume &amp; Portfolio Builder is an intelligent web application that automatically generates professional resumes and personal portfolio content using Meta LLaMA (Large Language Model). The system helps students and job seekers create industry-ready resumes quickly by providing AI-generated summaries, structured formatting, and downloadable outputs.

Users enter their basic details such as name, email, and skills. The application sends this information to the LLaMA model, which generates a professional resume summary and portfolio description. The content is then formatted into selected resume templates (Classic, Modern, Professional, Creative) and can be downloaded as a PDF or HTML file.

This project simplifies resume creation and enhances job readiness using Generative AI.
